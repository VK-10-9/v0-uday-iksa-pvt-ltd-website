# Project Overview: UDAY IKSA Website

## Introduction
Uday IKSA Pvt Ltd is a leader in sustainability engineering, specializing in natural daylighting and ventilation systems. This project is a high-performance web application designed to showcase their industrial innovations with a premium, technical aesthetic.

## Technical Stack
- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Language**: JavaScript / React
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **UI Components**: Custom-built based on Radix UI / Shadcn patterns.

## Core Philosophy
The website follows a **"Premium Industrial Aesthetic"**. This means:
- **Dark Mode First**: Deep slate backgrounds (`slate-950`) to provide high contrast for technical diagrams.
- **Accented Clarity**: Strategic use of `brand-orange` and `brand-red` to highlight key technical data and calls to action.
- **Technical Typography**: Heavy weights, wide letter tracking for labels, and condensed headings to mimic architectural blueprints and industrial branding.
- **Micro-Animations**: Subtle motion that suggests mechanical precision.

## Key Features
- **Dynamic Product Pages**: Highly detailed pages for NANOSUN, NANOVENT, and SKYTUBE with technical matrices and exploded anatomy views.
- **Sectoral Portfolio**: Categorized project showcase (Industrial, Commercial, PSU, etc.).
- **Performance Optimized**: Using `next/image` for large industrial installation renders and optimized component loading.
- **SEO & Legal**: Comprehensive metadata for industrial keywords and fully implemented privacy/terms/governance pages.
